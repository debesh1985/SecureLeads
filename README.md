# SecureLeads

SecureLeads is a React + Vite single-page application for WealthStem.ca, focused on high-trust mortgage, insurance, and investment planning experiences for Canadian families and business owners.

## Tech stack

- React 18
- Vite 5
- React Router
- Tailwind CSS
- Supabase JS

## Getting started

### Prerequisites

- Node.js (LTS recommended)
- npm

### Install

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## Project structure

- `src/` - Application source (routes, components, styles)
- `supabase/` - Supabase configuration

## Notes

- Update routes in `src/App.jsx` to add or modify pages.
- Tailwind configuration lives in `tailwind.config.js`.
